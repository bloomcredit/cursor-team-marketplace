# Caveman

Ultra-compressed communication mode for Cursor. Cuts ~75% of output tokens by speaking like caveman while keeping full technical accuracy.

Based on [JuliusBrussee/caveman](https://github.com/JuliusBrussee/caveman) (MIT license).

## Skills

- **caveman** — main mode with intensity levels: lite, full (default), ultra, wenyan
- **caveman-commit** — terse commit messages in Conventional Commits format
- **caveman-review** — one-line PR review comments with severity and fix
- **caveman-help** — quick-reference card for all modes and commands
- **compress** — compress markdown memory files into caveman-speak to save ~46% input tokens

## Rules

- **caveman.mdc** — always-on rule that activates caveman mode for every response

## Usage

Trigger with "caveman mode", "talk like caveman", or "less tokens please".

Switch intensity: `/caveman lite|full|ultra|wenyan`

Stop with: "stop caveman" or "normal mode"
